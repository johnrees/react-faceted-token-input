React Faceted Token Input

Tokenized input with faceted behavior for React

See it on Github

Features

  • Completely customizable
  • Keyboard selectable tokens
  • Support for rtl languages
  • Support for screen readers

Demo

How to use

Select the desired tokens by typing in the input one of the available tokens

Available tokens:

  • Bill Gates
  • Dennis Ritchie
  • James Gosling
  • Ken Thompson
  • Steve Jobs
